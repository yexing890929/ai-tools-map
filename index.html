编程 & 开发
CODING / DEV TOOLS
  • Cursor AI 原生代码编辑器，可直接对话修改代码库，适合全栈开发
  • Claude Code 终端 CLI 工具，理解整个项目上下文，适合复杂工程任务

  • Windsurf 支持多步骤 Agentic 编码，可自主完成功能模块
  • Bolt / v0 浏览器内直接生成 UI 组件或完整 Web 应用零代码
🇨🇳 国产工具
  • TRAE 字节跳动出品 AI IDE，深度集成豆包模型，免费使用字节

  • CodeGeeX 智谱 AI 开源代码模型，支持中文注释与代码互转智谱
🛠 API / 调试平台
  • Google AI Studio Gemini 模型调试与 API 接入平台，支持 Prompt 测试、导出代码，非开发者也可体验免费

思考 & 深度分析
REASONING / ANALYSIS
  • Claude 长上下文、细致推理，擅长复杂问题拆解、方案权衡
  • o1 / o3 OpenAI 推理模型，思维链更强，适合数学、逻辑、策略
  • Gemini 2.5 Pro Google 推理模型，支持超长上下文，适合大文档分析
  • NotebookLM 上传资料后深度问答，可生成播客摘要，学习研究利器
  • Deep Research Claude / Gemini / GPT 均有，自动搜索并生成深度报告
🇨🇳 国产工具
  • DeepSeek R2 深度求索出品，推理能力对标 o1，开源可本地部署深度求索
  • Kimi 月之暗面出品，超长上下文 200 万 token，适合大文件分析月之暗面
  • 腾讯元宝 接入 DeepSeek + 混元，支持深度搜索与长文总结腾讯
